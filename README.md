HF MACROECONOMIC SPILLOVERS

Goal

This code scarpes announcement records from www.forexfactory.com and evaluate the response of financial markets to macroeconomic announcements, in order to evaluate macroeconomic spillovers.

Content

Step 1: scrape entries, relevance and topics from the open-source macroeconomic calendar on www.forexfactory.com
  
Step 2: merge concurrent announcements to form tight high-frequency windows

Step 3: exploit high-frequency financial data to compute the response of financial assets within high-frequency windows 

Step 4: evalutate macroeconomic spillovers (e.g. impact of US macroeconomic data onto Treasury yield and Bund yields)

Step 5: evalutate spillovers generated by inflation data

Step 6: evalutate spillovers generated by growth data

Step 7: display historical evaluation across Governing Councils
