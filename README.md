# HF MACROECONOMIC SPILLOVERS
## Goal
This code scarpes announcement records from www.forexfactory.com and evaluate the response of financial markets to macroeconomic announcements, in order to evaluate macroeconomic spillovers.
## Content
1. Scrape entries, relevance and topics from the open-source macroeconomic calendar on www.forexfactory.com
2. Merge concurrent announcements to form tight high-frequency windows
3. Exploit LSEG high-frequency financial data to compute the response of financial assets within high-frequency windows 
4. Evalutate macroeconomic spillovers (e.g. impact of US macroeconomic data onto Treasury yield and Bund yields) ![alt text](https://github.com/GiulioCarloVenturi/HF-Macroeconomic-Spillovers/blob/main/Total_Hist.png)
5. Evalutate spillovers generated by inflation data ![alt text](https://github.com/GiulioCarloVenturi/HF-Macroeconomic-Spillovers/blob/main/Total_Hist.png)
6. Evalutate spillovers generated by growth data ![alt text](https://github.com/GiulioCarloVenturi/HF-Macroeconomic-Spillovers/blob/main/Total_Hist.png)
7. Display historical evaluation across Governing Councils ![alt text](https://github.com/GiulioCarloVenturi/HF-Macroeconomic-Spillovers/blob/main/Total_Hist.png)
